# JSX

[React Docs](https://beta.reactjs.org/learn)

1. JSX tags must be closed like `<br />`

## Writing components with JSX

1. Component names start with a captical letter, while HTML tags must be lowercase
1. Components cannot return multiple JSX tags, warp them into a Fragment like

   ```jsx
   function AppPage() {
     return (
       <>
         <h1>App Page</h1>
         <p>Welcome to the App</p>
       </>
     );
   }
   ```

1. Nesting components

   ```jsx
   function MyButton() {
     return <button>A Button</button>;
   }

   function MyParagraph() {
     return (
       <>
         <h1>Welcome</h1>
         <MyButton />
       </>
     );
   }
   ```

1. HTML class attribute is specified with `classname`

   ```jsx
   <img className="avatar" />
   ```

1. Quoted contents are passed as strings, JavaScript expressions can be
   written in JSX curly braces

   ```jsx
   return <img className="avatar" src={user.imageUrl} />;
   ```

### Conditional rendering

1. Use regular JavaScript logic to conditionally render contents

   ```jsx
   let content;
   if (isLoggedIn) {
     content = <AdminPanel />;
   } else {
     content = <LoginForm />;
   }
   return <>{content}</>;
   ```

### List rendering

Use `array`'s `map()` function to render lists of components
